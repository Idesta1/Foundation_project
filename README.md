# HYF Foundation team project - Memory Card Game

This is a README file for the Memory Card Game project for HYF foundation program.

# About Memory Card Game

This is a fun interactive memory card game built using HTML, CSS, and JavaScript. The objective of the game is to match pairs of cards by

flipping them over and remembering their positions. The game is designed to improve memory skills and provide entertainment.

## Features

- Responsive design for various screen sizes
- Smooth card flip animations
- Score tracking based on the number of attempts
- Restart game option to play again
- Shuffle cards after restart to randomize card positions
- Match detection logic
- Timer to track how long it takes to complete the game
- Simple and clean user interface
